# Enhanced Factor-Augmented Inflation Forecasting Framework

An advanced, central bank–grade inflation forecasting system developed by **Chirume Admire Tarisirayi**.

This framework integrates **principal component analysis (PCA)**, **econometric modelling**, and **machine learning techniques** to produce robust, policy-relevant inflation forecasts and scenario-based simulations.

---

## 🚀 Overview

Traditional inflation models often struggle with the following;
- High-dimensional macroeconomic data  
- Structural instability  
- Limited predictive performance  

This framework addresses these limitations by combining;
- Factor-based dimensionality reduction  
- Multi-model machine learning forecasting  
- Scenario analysis and policy simulation  

---

## ⚙️ Key Features

- 📊 **PCA-Based Factor Extraction**  
  Reduces large macroeconomic datasets into interpretable latent factors  

- 🤖 **Multi-Model Forecasting Engine**  
  Combines:
  - Elastic Net  
  - Random Forest  
  - Support Vector Regression (SVR)  
  - XGBoost  

- 📈 **Inflation Nowcasting Capability**  
  Uses high-frequency indicators to estimate current-period inflation  

- 🔄 **Scenario Analysis Engine**  
  Simulates shocks in:
  - Exchange rate  
  - Money supply  
  - Fiscal conditions  
  - Fuel and external prices  

- 🧠 **Regime Classification Framework**  
  Classifies inflation regimes aligned with SADC macroeconomic convergence targets  

- 📉 **Interactive Dashboard (Streamlit)**  
  Visualises forecasts, factor contributions, and scenario outcomes  

---

## 🧠 Methodological Framework

The model is based on a **Factor-Augmented Approach**, where inflation is driven by latent macroeconomic factors:

\[
\pi_t = \alpha + \beta F_t + \varepsilon_t
\]

Where:
- \( \pi_t \): Inflation  
- \( F_t \): Extracted macroeconomic factors (via PCA)  
- \( \beta \): Factor loadings  
- \( \varepsilon_t \): Error term
